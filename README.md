#  SPKanban

**SPKanban** — система управления проектами, реализованная на базе методологии Kanban. Приложение помогает организациям распределять задачи, контролировать этапы выполнения и выстраивать прозрачную коммуникацию между менеджерами и сотрудниками.

---

##  Основные возможности

-  **Kanban-таблица** — визуальное отображение задач и их статусов в реальном времени
-  **Ролевая модель доступа** — чёткое разделение прав для администратора, менеджера и сотрудника
-  **Встроенная система комментариев** — обратная связь, уточнения и замечания по каждой задаче
-  **Контроль качества** — обязательная проверка задач менеджером перед финальным завершением
-  **Хранение данных** — надёжная интеграция с Microsoft SQL Server

---

##  Роли и функционал

###  Администратор

- Просмотр полного списка учётных записей из БД
- Кнопки управления: `Добавить сотрудника`, `Выйти`
- Кнопка `Редактировать` (в строке таблицы пользователя) открывает ту же форму, что и создание, но с предзаполненными данными для быстрого изменения
- Полный контроль над учётными записями команды

###  Менеджер

- Создание задач и назначение ответственных сотрудников
- Управление жизненным циклом задачи по статусам:  
  `В работе` → `На проверке` → `Готово`
- Проверка задач, переведённых в статус `На проверке`:
  -  Одобрение → статус меняется на `Готово`
  -  Возврат на доработку → статус сбрасывается на `В работе`
- Работа с комментариями: чтение сообщений сотрудников, ответы, добавление замечаний при возврате задачи

###  Сотрудник

- Видит только задачи, назначенные лично ему
- После выполнения переводит задачу в статус `На проверке`
- Если менеджер возвращает задачу на доработку, сотрудник исправляет замечания и повторно отправляет её на проверку
- Комментирует задачи для уточнений, отчётов и просмотра инструкций менеджера

|-----------|------------|

| **UI Framework** | WPF (Windows Presentation Foundation) |
| **Язык** | C# |
| **База данных** | Microsoft SQL Server |

##  Требования

- **ОС**: Windows 10/11
- **.NET**: .NET Framework 4.7.2+ или .NET 6.0+
- **СУБД**: Microsoft SQL Server 2016+
- **IDE**: Visual Studio 2019/2022 (рекомендуется)
